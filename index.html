An Implementation of Bubble Cursor from Microsoft

Point Cursor

Standard point cursors have one activation point.

As applications become more complex and user interfaces more difficult to navigate, pointer cursors signficantly degrade software usability.

Bubble Cursor

The bubble cursor from Microsoft is one solution.

It changes its activation area to select the nearest widget which makes software more usable by increasing a target's effective width.
